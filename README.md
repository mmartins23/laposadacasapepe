# La Posada Casa Pepe - Restaurant Website 🍽️

This repository contains the code for the official website of **La Posada Casa Pepe**, a restaurant located in Abanilla, Murcia, known for its traditional Spanish dishes, including *paella* and *carne a la brasa*. The website provides an inviting digital experience to help customers explore the menu, learn about the restaurant, and make reservations.

## 🌐 Website

**[La Posada Casa Pepe](https://www.laposadacasapepe.com/)**

## 📋 Features

- **Dynamic Menu Display**: A responsive layout showcasing the full restaurant menu with highlighted recommendations.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Easy Navigation**: A user-friendly, responsive navigation bar for seamless access to all sections.
- **Contact Information**: Essential contact details and an embedded Google Map for easy location finding.
- **Team Section**: An overview of the dedicated staff, enhancing the welcoming atmosphere of the restaurant.
- **Reservation Options**: Quick access to reserve a table through a contact form and social media links.

## 🖥️ Tech Stack

- **Front-end**: React, JavaScript, HTML, CSS
- **Styling**: Flexbox for responsive layout, custom CSS styles for a unique and inviting look
- **SEO Optimized**: Structured to improve discoverability for relevant keywords, such as "paella" and "carne a la brasa."

## 🚀 Getting Started

To run the project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/la-posada-casa-pepe.git
   ```
2. **Navigate into the directory**:
   ```bash
   cd la-posada-casa-pepe
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm start
   ```
