# La Posada Casa Pepe - Website

Este repositório contém o código-fonte do site do restaurante **La Posada Casa Pepe**, um local histórico em Abanilla, Murcia, conhecido por sua culinária tradicional e ambiente acolhedor.

## Visão Geral

O site da **La Posada Casa Pepe** foi criado para proporcionar aos visitantes uma visão completa sobre o restaurante, desde sua história até os menus detalhados e os canais de contato. O site é composto por várias seções interativas que informam e encantam os usuários, oferecendo uma experiência rica sobre a gastronomia e o ambiente do restaurante.

### Funcionalidades:

- **Navbar:** Menu de navegação com links para diferentes seções do site.
- **Hero:** Apresenta o nome do restaurante e opções rápidas para visualizar o menu e fazer reservas via WhatsApp.
- **Welcome, About, Gastronomy, Team, Menu e Contact:** Seções com informações detalhadas sobre o restaurante, equipe, pratos e opções de contato.
- **Footer:** Links para redes sociais e localização no Google Maps.

## Componentes

Os principais componentes usados no projeto estão listados abaixo:

- `Navbar`: Barra de navegação com ícones e links de rolagem suave.
- `Hero`: Seção principal com destaque para o nome do restaurante e links rápidos.
- `Welcome`: Seção de boas-vindas com notícias e atualizações.
- `Title`: Componente reutilizável para títulos e subtítulos.
- `About`: Informações sobre a história do restaurante.
- `Gastronomy`: Galeria de pratos organizados por categoria.
- `Menu`: Apresentação dos menus oferecidos no restaurante.
- `Team`: Seção destacando a equipe e o compromisso com a qualidade.
- `Contact`: Seção de contato com integração com Google Maps e redes sociais.
- `Footer`: Rodapé com links para redes sociais e informações de copyright.

## Instalação

Para rodar o projeto localmente, siga os seguintes passos:

1. Clone este repositório:
   ```bash
   git clone https://github.com/usuario/repositorio.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd repositorio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

## Como Executar

Para rodar o projeto localmente, utilize o comando:

```bash
npm start
```

Isso iniciará o servidor de desenvolvimento e o site estará acessível em `http://localhost:3000`.
